# Game Modules

Bu klasor oyun domain kodlari icindir. Discord komutlari ve REST API controllerlari burada is kurali yazmaz; bu klasordeki servisleri cagirir.

Planlanan modul sinirlari:

- `players`: oyuncu profili, XP, level ve para birimi.
- `pets`: baslangic peti, pet statlari, rarity, mood ve passive ability.
- `inventory`: materyal, consumable, equipment ve pet envanteri.
- `professions`: madencilik, balikcilik, odunculuk, ciftcilik ve yemek pisirme.
- `equipment`: ekipman takma, upgrade, tamir ve hasar kurallari.
- `economy`: gold, gem, sans oyunlari ve transactionlar.
- `quests`: gunluk/haftalik gorevler ve basarimlar.
- `market`: oyuncu-oyuncu satis ve takas akislari.
- `events`: haftalik etkinlik, boss/event ve sezon temeli.
- `battles`: sonraki faz pet savaslari.
